# SpecifyPlus Kit Installation

## What is SpecifyPlus Kit?

**SpecifyPlus Kit** is an AI-enhanced design system and component library that bridges design tools with code generation.

## Installation Methods

### Method 1: Quick Start (Recommended)
```bash
# Create new project with SpecifyPlus
npx create-specifyplus-app my-ai-project

# Or add to existing Docusaurus project
cd physical-ui-textbook
npm install @specifyplus/kit @specifyplus/cli @specifyplus/react
