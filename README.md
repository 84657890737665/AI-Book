# Physical AI & Humanoid Robotics Textbook

Welcome to the Physical AI & Humanoid Robotics textbook project! This comprehensive guide bridges the gap between artificial intelligence and physical robotics.

## About

This Docusaurus project contains educational materials for learning Physical AI concepts, from ROS 2 fundamentals to deploying AI models on real humanoid robots.

## Prerequisites

- Node.js version 18 or higher
- npm or yarn package manager

## Installation

1. Navigate to the website directory:
```bash
cd website
```

2. Install dependencies:
```bash
npm install
```

## Local Development

1. Run the development server:
```bash
npm run start
```

This command starts a local development server and opens the website in your browser. Most changes are reflected live without having to restart the server.

## Build

To build the static website for production:

```bash
npm run build
```

The static files will be generated in the `build` directory.

## Deployment

The website can be deployed to various platforms:

- GitHub Pages
- Netlify
- Vercel
- Traditional web servers
